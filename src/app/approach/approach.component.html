Approach

The first version of the website is made in the following order.

  1. Have an idea.
  2. Define requirements through MoSCoW.
  3. Setup local web server.
  4. Add git repository hosted on Github.
  5. Add HTML.
  6. Add CSS through Sass.
    1. Add taskrunner.
  7. Review
    1. Review by others, increases quality.
